E-commerce Promotional Email Template: RYVO Summer Sale

This folder holds the high-fidelity design mockups for a promotional email template for the RYVO E-commerce brand's "Summer Sale 2025."

Goal: To design a visually appealing and responsive email that drives traffic to the sale page and encourages immediate action using a clear discount code (SUMMER20).
Key Focus Areas:

Responsiveness: Designing for optimal viewing on both desktop and mobile devices.

Visual Appeal: Utilizing brand colors and clean hierarchy to highlight the main offer.

Clear CTA: Ensuring the "Claim Offer" button is prominent and actionable.
